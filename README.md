# csv-analyzer
Приложение для анализа CSV-файлов на Streamlit 
### Описание

Программа предназначена для анализа табличных данных из CSV-файлов. Она позволяет загрузить файл, автоматически определить типы столбцов (числовые, даты, текстовые), выполнить статистический анализ и построить графики.

**Алгоритмы и формулы:**

- Определение типа столбца основано на проверке возможности преобразования данных в числовой формат (через `pd.to_numeric`) или в дату (через `pd.to_datetime` с проверкой паттернов дат).

- Статистический анализ для числовых столбцов включает расчёт:
  - Среднее арифметическое: `x̄ = (Σ xi) / n`
  - Медиана - центральное значение упорядоченного ряда
  - Стандартное отклонение: `σ = √(Σ(xi - x̄)² / n)`

- Для графиков используется библиотека Plotly с построением:
  - Линейных графиков для отображения трендов
  - Точечных диаграмм (scatter plot) для выявления корреляций
  - Столбчатых диаграмм для сравнения категорий
  - Гистограмм и box-plot'ов для анализа распределения данных

### Установка зависимостей

Для запуска программы необходим Python версии 3.8 или выше.

1. Склонируйте репозиторий или скачайте файл `app.py`
2. Установите зависимости через pip:

```bash
pip install streamlit pandas numpy plotly openpyxl kaleido
```
Или используйте файл requirements.txt:  
```bash
pip install -r requirements.txt
```

### Запуск  
Запустите приложение из терминала, находясь в папке с файлом app.py:  
```bash
streamlit run app.py
```
После запуска автоматически откроется браузер с приложением по адресу http://localhost:8501.

### Порядок работы:  

- Загрузите CSV-файл через боковую панель  

- На вкладке "Данные" просмотрите таблицу и информацию о столбцах  

- На вкладке "Статистика" выберите столбец для анализа  

- На вкладке "Графики" постройте визуализацию зависимостей
